# 🏠 House Price Prediction

A simple Machine Learning project that predicts apartment prices based on apartment features such as area and number of rooms.

## 📌 Project Overview

This project uses a Machine Learning regression model to estimate apartment prices from a housing dataset.

The model predicts apartment prices using:

- 📐 Apartment Area (Square Meters)
- 🛏️ Number of Rooms

## 📊 Dataset

The dataset contains apartment information including:

- Area
- Number of Rooms
- Price

The data is cleaned and prepared before training the model.

## 🤖 Model

The model is trained using supervised Machine Learning techniques to predict apartment prices.

### Steps

1. Load the dataset.
2. Preprocess the data.
3. Split the data into training and testing sets.
4. Train the regression model.
5. Evaluate the model.
6. Predict apartment prices.

## 🛠 Technologies

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Jupyter Notebook

### Input

- Area: 120 m²
- Rooms: 3

### Output

Predicted Price: 2,350,000 EGP
